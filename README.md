# arquitetura-software
Arquitetura de software criada no Antigravity

Este repositório contém o arquivo `.resolved` da arquitetura de software criada no Antigravity.

## Como visualizar

- Abra o arquivo `arquitetura.resolved` no Antigravity para ver o diagrama completo.

## Como contribuir

1. Faça um **Fork** do repositório.
2. Abra o arquivo `.resolved` no Antigravity e edite o que for necessário.
3. Salve e envie um **Pull Request** com as alterações.
4. Use a aba **Issues** para sugerir melhorias ou apontar problemas.
